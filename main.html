1. Topologies des réseaux

La topologie décrit la structure d’un réseau.

Bus

Un seul câble partagé par tous les ordinateurs.

Anneau

Les ordinateurs forment une boucle fermée.

Étoile

Les postes sont reliés à un équipement central.

2. Équipements réseaux

  • Hub : diffuse à tous
  • Switch : envoie au bon destinataire
  • Routeur : relie les réseaux
  • Serveur : fournit des services

3. Modèle OSI

  1. Physique
  2. Liaison
  3. Réseau
  4. Transport
  5. Session
  6. Présentation
  7. Application

4. Adressage IP

Une adresse IP est composée d’un NetID et d’un HostID.

5. Masque de sous-réseau

Il permet d’identifier la partie réseau.

6. CIDR

Exemple : 192.168.1.0/24

7. Balises HTML

  • head : informations
